# DriverRW

## 项目介绍

本项目是一个基于Windows7 x64操作系统的驱动读写项目，主要用于实现应用层与驱动层之间的通信，实现的功能包含内存读写，进程监控，模块管理和保护功能……该项目展示了多种Windows内核编程技术：内存操作，系统回调注册，进程挂靠，特征码搜索等内核开发技巧。
